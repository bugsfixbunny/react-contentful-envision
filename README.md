## Prerequisites
- [ ] NodeJS (version >=10.13.0)
- [ ] NPM
- [ ] Contentful API Access
- [ ] An IDE of your choice ([Visual Studio Code](https://code.visualstudio.com/) recommended)

<a name="setup"/></a>
## Setup
1. `git clone git@github.com:LMLI-Cohort-9/Envision-2040.git`
2. `cd Envision-2040/`
3. `npm install`
4. `nmp run setup`
5. `npm run dev`
